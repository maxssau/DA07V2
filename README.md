# DA07V2
DA07V2 DAC firmware

Used devices:

MCU STM32F446RC
DSP ADAU1701
LCD: TFT 1.8" 128x160 px, 16bit dept, SPI bus
Rotary encoder with button
DAC: CS43198
Vol Ctrl: PGA2311
